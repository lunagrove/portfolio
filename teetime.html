Tee Time

A mobile application for Android and iOS that allows event creation, management and reporting for hosts, and event registration for users.
Built with React Native, using AWS API Gateway, lambda functions, Amplify, Cognito, and a CockroachDB relational database.

User authentication handled by Amplify and Cognito, featuring email confirmation, password reset and forgot password functionality. All api endpoints authorised with tokens.

User upcoming events page. Icons indicate status of event in relation to this user - registered, waitlisted, eligible, ineligible. Event capacity and number of spots taken is shown.

Event details page for a registered event. User is able to view the generated QR code, or withdraw from the event.

QR code is encoded with the user id and the event id. Could be scanned at the tournament and used to update the attendance status in the application. Currently still a manual process for hosts.

Event details page for a waitlisted event. Shows the user's position in the waitlist queue. User is able to remove themselves from the waitlist if required.

Calendar view of a user's upcoming events, using the same color coding for events: green for registered, orange for waitlisted, red for ineligible. Event details page can be accessed by tapping an event.

Main menu for host functionality: details about any event starting today, ability to create an event, ability to list events, ability to list users who have signed up for the app.

Event creation page: events are assigned an eligibility type of guest list (host selects users who can register), bronze, silver or gold tier (has to match or be exceeded by the user's membership tier), or loyalty (user must have attended x number of previous events).

Past events page. These events are completed and show the number of users who registered. Tapping on a past event will allow viewing of the event details page.

Event details page for a completed event shows attendance figures captured so far, and allows the attendance page to be viewed.

Attendance page shows all users who registered for this event and allows the host to flag them as having attended or recording them as a no show.

Users list page showing all users currently signed up for the app. Icons indicate which membership tier each user has: gold, silver or bronze. Tapping on a user brings up the user details page.

User details page allows the membership status of a user to be changed. Their event attendance history can also be viewed.

User event attendance page shows past events this user registered for, with filters by attended or no show.
