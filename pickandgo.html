PickAndGo

A fully responsive web application for restaurants that allow customisation of products on their menu.
Built with C# ASP .Net Identity Framework, using a Microsoft SQL Server database, and implementing Paypal for payments and SendGrid for confirmation emails.

Landing page, encouraging the user to start an order.

The user may already have an account, might want to sign up for an account, or might just want to continue as a guest.

Log in page, handled by ASP .Net Identity Framework, with email confirmation, password reset and forgot password functionality.

Initial page for a user with an account. Invites them to start a new order or take a look through their favourites to find something they have tried before.

Sandwich selection page: three sandwich sizes to choose from.

Customize page: all ingredients available are shown by category. A selection from the breads category is required, everything else is optional. Each sandwich has a base price associated with its size.

The user can customize all ingredients, by selecting either the regular quantity for that size sandwich, or extra. The price is adjusted accordingly using the base price for each ingredient.

Shopping cart: each item in the cart can be edited, which returns to the customize page for changes, added to the order again if a second sandwich the same is required, or deleted from the cart.

A user with an account can view his previous purchases. Sandwiches flagged as favourites are shown and new ones can be added.

When adding a sandwich to favourites, a name must be given to it, making it easier to find in the future.

A user with an account can view his favourites page. Sandwiches can be removed from this list by tapping the red heart. The name of each favourite can be edited.

Changing the name of a favourite.

About page, with a catchy phrase, image and logo. Scrolling down reveals a description of the restaurant and its services, and a map showing its location.
